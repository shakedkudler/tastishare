# Tastishare

Tastishare is a recipe sharing platform designed to help users discover, share, and manage their favorite recipes. The project aims to provide an intuitive interface for browsing, searching, and contributing recipes, making cooking and meal planning easier for everyone.

## Features

- User-friendly recipe submission and editing
- Search and filter recipes by ingredients, cuisine, or category
- Save favorite recipes for quick access
- Responsive design for desktop and mobile devices

## Getting Started

Clone the repository and follow the setup instructions to run the project locally.
